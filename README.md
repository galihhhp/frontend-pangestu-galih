# Cara Install & Menjalankan Project

1. **Clone repo ini:**

   ```bash
   git clone <url-repo-anda>
   cd <nama-folder-repo>
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # atau
   yarn install
   ```

3. **Jalankan development server:**

   ```bash
   npm run dev
   # atau
   yarn dev
   ```

   Buka browser ke [http://localhost:5173](http://localhost:5173)
